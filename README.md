# BPSC Revision Tracker

A 120-day BPSC Prelims revision tracking PWA — optimized for iPad.

## Features
- ✅ Spaced repetition: R1(+1d) → R2(+3d) → R3(+1w) → R4(+2w) → R5(+1m) → R6(+75d)
- 🔴 Red highlight for overdue/due-today revisions
- ↩ Undo / ↪ Redo — full history (50 steps)
- 📊 Statistics: heatmap, section progress, streak
- 💾 IndexedDB storage + localStorage backup
- ⬇ Export / ⬆ Import JSON (never lose data)
- 📱 PWA — install on iPad from Safari → Share → Add to Home Screen
- 🌐 Works fully offline after first load

## Deploy on GitHub Pages
1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Source: main branch, root `/`
3. Your tracker will be live at `https://yourusername.github.io/reponame`

## How to Use
- **Today view**: Shows all due/overdue revisions with one-tap completion
- **All Days**: Browse by section, tap any card to expand and manage
- **Stats**: Track progress, streaks, heatmap
- **Tick any checkbox** → date picker appears → confirm
- **Tap a done checkbox** → option to remove (undo is always available)
